This app is an authentication system using Firebase for future projects. 
It allows users to login using Google's login as well as registering a 
new email and password. The frontend uses React, NextJS, NodeJs, and the backend will be 
using MongoDB. 

The source for most of the functionality has been from various tutorials.
Ryan Dhungel's tutorials in particular have been integral. You can find
out more about his by visiting his website https://kaloraat.com/
